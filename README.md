## ✨ Quick start

1.  **Clone this repository.**

    ```sh
     https://github.com/ferhatkefsiz/real-estate-website.git
    ```

2.  **Install Dependencies**

    ```bash
        $ npm install
    ```

3.  **Runs the app in the development mode.**


    ```bash
        $ npm start
    ```

     Open http://localhost:3000 to view it in the browser.
